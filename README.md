# Mitocondria Lab - Tienda de Péptidos

Sitio web para Mitocondria Lab, especializado en péptidos de investigación de alta calidad.

## Estructura

- `index.html` - Página principal
- `/productos/` - Páginas de detalle de productos
- `/images/` - Imágenes de productos
- `favicon.svg` - Logo

## Deploy

Este sitio está hosteado en Vercel. Cualquier push a main se despliega automáticamente.

```bash
git push origin main
```
